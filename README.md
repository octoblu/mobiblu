# Mobiblu

## Installation

1. `npm install steroids bower -g`
2. `git clone http://github.com/octoblu/mobiblu.git`
3. `cd mobile`
4. `bower install`
5. `steroids connect`
    - Type `s` to launch simulator
    - Press `[enter]` to re-compile


## Deploy

Commit code first

1. `steroids deploy`
2. Go to build.appgyver.com and build the distributions you want
